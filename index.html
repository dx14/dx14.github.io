Glenna

Glenna is my first iOS app that I created in a team of three. It functions as a location-based virtual bulletin board, using the GoogleMaps API to allow users to post, edit, delete and subscribe to events and alerts posted by nearby users. The app also utilizes a simple authentication system for log-in and user creation, as well as the ability to filter, favorite, and sort events.

Built using Swift/Xcode, Node.js, and Heroku.

VOOGASalad

VOOGASalad is an acronym for "Video Object Oriented Game Architecture Salad." This project is based off Game Salad, a game creator and engine that allows users to create a game from scratch and play it. I worked with a team of 10 others in creating our own turret-defense genre game player, engine, and editor. I worked on the game's authoring environment, which allowed players to create and customize the elements of their game, including units and their stats, the game board, and game levels, and also save and load existing games. Combined with the game player and engine, our program allowed for the making of various types of turret defense games.

Built using Java, JavaFX, and XML.

Source code can be found here.

Cell Society

Cell Society is a simulator I developed in a team of 3 that is able to visualize the transitions of gridded cellular automata. Cellular automata are grids of colored cells that change behavior and properties depending on the state of its neighboring cells. Our simulator is able to mimic relationships such as segregation, predator-prey, and the spreading of fire.

Built using Java, JavaFX, and XML.

Source code can be found here.

Under the Sea

Under the Sea is the first program I developed that includes front-end programming. It is a single-player turn-based strategy game that allows for simple open world exploration and battle phases. The game utilizes non-playable characters as enemies and random number generators to enhance the unpredictability of each playthrough.

Built using Java and JavaFX.
